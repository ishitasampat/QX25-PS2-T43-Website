ABOUT

Find My Fund is an intelligent AI-powered platform that helps users discover the most relevant mutual funds, stocks, or ETFs using natural language queries. Whether you're typing "tax saving SBI fund" or "tech sector above 1000Cr AUM", our system understands your intent even with typos or partial input and instantly matches it with accurate investment options.

Built for the modern investor, Find My Fund combines the power of semantic search, metadata filtering, and a lightning-fast AI model to simplify investment discovery like never before.

More..

Instead of relying on exact matches, the platform understands what you're trying to say even with typos or vague inputs and returns the most relevant investment options. Powered by the MiniLM-L6-v2 model and FAISS vector search, it processes queries semantically and uses metadata like sector, fund house, and AUM to refine results, making investment discovery faster, smarter, and more intuitive.

Getting Started

To begin using Find My Fund, simply enter your investment-related query in plain English no technical terms needed. Whether you're looking for "tax-saving funds from SBI" or "tech sector mutual funds above 1000Cr," the app instantly interprets your intent using AI and returns the most relevant matches. You can use the web version or download the app, explore suggested funds, and refine your search using filters like sector, fund house, and AUM. No logins, no confusion just smart, intuitive fund discovery at your fingertips.

Advanced Features

Find My Fund goes beyond basic search by offering advanced capabilities like semantic understanding of user queries through AI-powered embeddings, typo and abbreviation tolerance, and intelligent filtering using metadata such as fund house, sector, AUM, and asset class. The system also supports fuzzy matching for partial queries, ranks results by relevance, and delivers real-time responses using a lightweight MiniLM-L6-v2 model. Designed for both speed and accuracy, it ensures every user—from beginner to expert—finds the perfect fund with minimal effort.

API Documentation

The Find My Fund API allows developers to integrate smart mutual fund discovery into their own platforms. Simply send a POST request to the /search endpoint with a user query string, and the API will return the most relevant funds based on semantic search and metadata filters. The API supports fuzzy matching, handles typos, and uses a pre-trained MiniLM-L6-v2 model for fast vector-based retrieval. Responses include fund name, category, AUM, fund house, and a confidence score. Ideal for fintech platforms, chatbots, or portfolio advisors looking to add intelligent investment suggestions.
